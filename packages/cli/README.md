## 📦 Installation

```bash
git clone https://github.com/hieudoanm/chess.engine.git
cd chess.engine/packages/cli
make build
sudo cp bin/engine /usr/local/bin/
```

## 🚀 Usage

```bash
engine
# Interactive REPL — try: help | e2e4 | print | quit
```
